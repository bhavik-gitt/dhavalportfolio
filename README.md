# Advanced Portfolio Website

Modern animated portfolio built using React, Vite, TypeScript, Tailwind CSS and Motion.

## Features

- Responsive Design
- Animated Hero Section
- Interactive Experience Timeline
- Certification Showcase
- Skills Dashboard
- Achievement Wall
- Contact Section
- Custom Cursor
- Smooth Scrolling
- Glassmorphism UI
- Real-Time Visual Effects

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Motion
- Material UI
- Radix UI
- Recharts

## Installation

npm install

## Run

npm run dev

## Build

npm run build

## Preview

npm run preview

## Deployment

Optimized for Vercel deployment.
