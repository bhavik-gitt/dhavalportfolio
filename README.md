This portfolio serves as a professional digital presence for **Dhaval Ratnottar**, highlighting career achievements, HR expertise, leadership capabilities, certifications, and professional journey through an engaging and interactive experience.

The project combines executive branding with modern web technologies to create a memorable first impression for recruiters, organizations, and business leaders.

# 🚀 Local Development

Clone the repository:

```bash
git clone https://github.com/bhavik-gitt/dhavalportfolio.git
```

Navigate to project:

```bash
cd dhavalportfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build project:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
